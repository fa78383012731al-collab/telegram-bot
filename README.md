# Order Management Telegram Bot

بوت إدارة طلبات لفريق الخدمات التعليمية.

## المتغيرات المطلوبة
- BOT_TOKEN
- NOTIFY_CHAT_ID
- ADMIN_IDS

## مثال ADMIN_IDS
8015617513,123456789,987654321

## التشغيل على Render
- Build Command: `pip install -r requirements.txt`
- Start Command: `python bot.py`

## ملاحظات
- استخدم Python 3.10
- ضع NOTIFY_CHAT_ID على رقم مجموعة الفريق، وليس رابط الدعوة
- يمكن استخدام /id داخل المجموعة لمعرفة رقمها
